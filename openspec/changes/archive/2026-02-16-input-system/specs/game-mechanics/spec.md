## MODIFIED Requirements

### Requirement: Controls

| 入力 | アクション |
|------|-----------|
| WASD | 移動 |
| マウス | エイム方向 |
| 右クリック | 通常攻撃 |
| Q | スキル1 |
| E | スキル2 |
| R | アルティメット（Lv3解放） |
| Space | ドッジダッシュ（長クールダウン） |
| 左クリック | スキルターゲット確定（Normal Cast モード） |

スキルターゲティングは2つのキャストモードをサポートする:
- **Normal Cast**: スキルキー押下でターゲティングモードに入り、左クリックで対象確定・発火、右クリックでキャンセル。
- **Quick Cast**: スキルキーを離した瞬間にマウス位置に向けて発火。押下中に右クリックでキャンセル。

#### Scenario: 右クリック通常攻撃
- **WHEN** プレイヤーがゲームプレイ中に右クリック
- **THEN** エイム方向に向けて通常攻撃が発火される

#### Scenario: Normal Cast スキルターゲティング
- **WHEN** プレイヤーが Normal Cast モードでスキルキー（Q/E/R）を押下
- **THEN** そのスキルのターゲティングモードが開始される
- **WHEN** ターゲティングモード中にプレイヤーが左クリック
- **THEN** クリック位置に向けてスキルが発火される
- **WHEN** ターゲティングモード中にプレイヤーが右クリック
- **THEN** ターゲティングがキャンセルされ idle に戻る

#### Scenario: Quick Cast スキルターゲティング
- **WHEN** プレイヤーが Quick Cast モードでスキルキー（Q/E/R）を離す
- **THEN** 現在のマウス位置に向けてスキルが発火される
